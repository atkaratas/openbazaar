# GÖREV: Temel Arayüz (UI) Bileşenlerinin Üretilmesi
**Hedef:** Shadcn UI ve Tailwind CSS kullanarak Pazar Yeri (Marketplace) için temel görsel bileşenleri (Components) oluşturmak.
**Gereksinimler:**
1. Global Navbar (Çoklu Dil/Para Birimi seçici dropdown, Arama Çubuğu, Sepet İkonu, Kullanıcı Menüsü).
2. Product Card (Ürün Kartı) bileşeni (Resim, İsim, Fiyat, Satıcı Logosu).
3. Footer (Bülten aboneliği, Linkler, Ödeme Yöntemi İkonları).
**İstenen Çıktı:** `components/ui` ve `components/layout` altına yerleştirilecek modüler React (TypeScript) bileşen kodları.
